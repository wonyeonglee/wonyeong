# 1. 앱 Kinteract
## 1.1 앱 설명
>Kinteract= I(In) + C(Class) = K + interact
즉, 수업 중에 소통한다는 의미인 inter 
>Kinteract의 이름에서 알 수 있듯이 In Class, 즉 수업 시간에 활용하거나 질문을 돕는 웹 채팅입니다.
